# vue2-eleme

> 基于vue-cli v2.8.1 构建饿了么webapp实例, 仅供参考

#### 技术栈

```
  vue2
  vuex
  vue-router
  axios
  postcss
  - autoprefixer
  - precss
  - postcss-import
```
#### postcss 说明
这里用precss替代less/sass/stylus, 感觉终于找到符合自己的css处理工具了, precss的语法(与sass语法相近)可以到上面链接中查看文档.
#### 运行程序
```
npm install
npm run dev
http://localhost:8080
```
